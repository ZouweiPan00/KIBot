# KIBot

A Knowledge Integration Agent for Multi-Textbook Compression.

KIBot is a web dashboard for integrating multiple textbooks into a compact, evidence-backed knowledge core with graph visualization, cross-textbook deduplication, RAG citations, teacher feedback, and report generation.

## Current Status

This repository currently contains the project design and implementation plan. Implementation will proceed inside this repository.
